# Follow Mode Implementation Defaults

Use the following default values for Follow Mode setup:

- Follow Roll start (deg): `15`
- Follow Roll mid (deg): `20`
- Deadband: `4`
- Expo curve: `100`
- Roll speed: `10`
- Pitch speed: `10`
- Yaw speed: `10`
- LPF: `4`
- Roll range (deg): `60`
- Pitch range (deg): `35`
- Yaw range (deg): `5`
